#Departmental Courses

A Django web application for managing and displaying university departmental courses across different semesters. Built to simplify course browsing and access for students and departments.
## Features

    Department and Semester filtering

    Add, view, and manage courses

    Simple and intuitive UI

    Django admin support

## Tech Stack

    Backend: Django

    Frontend: HTML, CSS, Bootstrap

    Database: SQLite (default)
##How to Run

    Clone the repository

    git clone https://github.com/Eduedics/departmentalCourses.git
    cd departmentalCourses
    
    Install dependencies
    
    pip install -r requirements.txt
    
    Run migrations
    
    python manage.py migrate
    
    Start the server

    python manage.py runserver

    Visit http://127.0.0.1:8000/ in your browser to explore the app.
